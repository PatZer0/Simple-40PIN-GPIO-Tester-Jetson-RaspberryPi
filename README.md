# 40PIN GPIO 快速测试程序
此程序将检查可用GPIO，并让可用的GPIO不断在高低电平之间切换。
适用于Jetson和树莓派，或修改GPIO库后适配其他的平台，如香橙派等。

# 硬件需求
最简单的方法，当然是使用万用表检测电压，便能够确定GPIO的状态。
此外，你还可以选择按照这个开源项目制作一个方便的GPIO测试器：https://oshwhub.com/patzer0/40pin-gpio-tester
